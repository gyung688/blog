폴더 구조가 수평으로 되어있으면 폴더가 많아질수록 보기가 불편해집니다.  

이럴 때 수직 구조로 변경하면 보기 편합니다.  

방법은 간단합니다.  
<br/>

```
Package Explorer 옆에 있는 세모 클릭 > Package Presentation > Flat
```
